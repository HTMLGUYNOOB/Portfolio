## Apospan Portfolio Project

A modern, bilingual Personal Portfolio designed with a focus on **UI/UX Excellence.** The project was developed starting from a blank canvas in Figma and evolved into a fully functional, dynamic website.

### 1. Design Phase (UI/UX)

ğŸ”¹**Figma Design:** Visual design and design system (layout, colors, typography) created for full UI control.<br>
ğŸ”¹ **Prototyping:** Flow creation to ensure seamless User Experience (UX) across desktop and mobile.

ğŸ”¹ **Breakdance Builder:** Used as the base for performance-oriented HTML markup.<br>
